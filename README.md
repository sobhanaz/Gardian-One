# Gardian One

A comprehensive smart security and surveillance system proposal — **نگاهبان** (Negahban)

## Overview

Gardian One is an intelligent security management system proposal that provides:

- 🔍 Real-time surveillance monitoring
- 📊 Smart analytics and reporting
- 👥 Personnel tracking and management
- 🎯 Intelligent threat detection
- 📱 Real-time notifications

## Contents

- `index.html` — Landing hub linking to every deliverable
- `negahban-proposal.html` — Interactive proposal document (main deliverable)
- `negahban-why.html` — Client-facing value & business case (why buy / why us)
- `negahban-onepager.html` — One-page executive summary
- `negahban-cost-calculator.html` — Interactive cost-structure calculator
- `negahban-roi.html` — Interactive ROI / break-even analysis
- `negahban-technical-spec.html` — Technical specification & implementation guide
- `negahban-supplement.html` — Financial, technical & business supplement (sensitivity, BMC, SWOT)
- `negahban-pitch.pptx` — Pitch deck (PowerPoint, downloadable)
- `negahban-proposal.pdf` / `.docx` — Exported proposal
- `content.js` — Single source of proposal content (used to generate HTML/Word output)
- `favicon.svg` — Brand shield favicon shared across pages

> **Note:** all financial figures assume a base USD rate of **160,000 toman** (دلار = ۱۶۰٬۰۰۰ تومان); change it live in the cost calculator.

## Viewing the Proposal

Open `index.html` (the hub) or `negahban-proposal.html` directly in a web browser. The proposal's table of contents links to every interactive and supplementary page.
