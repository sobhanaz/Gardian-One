## 📋 نمای کلی

**نگاهبان** (Negahban - "The Guardian") یک پلتفرم جامع **انسان + ماشین** در حوزه امنیت است که امنیت فیزیکی را با تقویت هوشمندانه نیروهای نگهبانی مستقر در محل متحول می کند، نه با جایگزینی آن ها.

### مسئله
امنیت سنتی معمولاً با این چالش ها روبه رو است:
- ❌ پایش دستی که در معرض خستگی و خطای انسانی قرار دارد
- ❌ واکنش دیرهنگام به رخدادها
- ❌ قابلیت محدود ردیابی لحظه ای
- ❌ نبود ردپای ممیزی یکپارچه و قابل اتکا

### راهکار
نگاهبان با ترکیب هوش مصنوعی پیشرفته و روش های اثبات شده امنیتی، این قابلیت ها را ارائه می دهد:

- 🧠 **تشخیص چهره مبتنی بر هوش مصنوعی** - تشخیص افراد مجاز/غیرمجاز به صورت لحظه ای همراه با هشدار فوری
- 👤 **اعتبارسنجی نگهبان** - تحلیل رفتاری برای اطمینان از حضور فرد صحیح در پست
- 📍 **موقعیت یابی داخلی زنده (RTLS)** - ردیابی زنده نگهبان و دارایی ها روی نقشه داخلی
- 🚪 **کنترل دسترسی هوشمند** - ترمینال بیومتریک، گیت و مدیریت دسترسی مبتنی بر نقش/ناحیه
- 📊 **ردپای ممیزی کامل** - ثبت هر رویداد با زمان، مکان و تصویر
- 🏛️ **حاکمیت داده** - استقرار 100٪ درون سازمانی؛ داده های بیومتریک هرگز از سایت خارج نمی شوند

## سناریوی مرجع

ارقام این اسناد بر اساس یک سناریوی نمونه تنظیم شده اند:

- **ساختمان 11 طبقه** ایمنی/اداری
- **500** کارمند + **200** مراجعه کننده روزانه
- حدود **110** دوربین، **18** ترمینال بیومتریک، **4** گیت

## برآورد هزینه در یک نگاه

> فرض پایه: **USD = ۱۶۰٬۰۰۰ تومان** (160,000 toman). همه اعداد به صورت زنده در
> [ماشین حساب هزینه](https://sobhanaz.github.io/Gardian-One/negahban-cost-calculator.html) قابل ویرایش هستند.

| آیتم | برآورد |
|---|---|
| استقرار هر ساختمان (سخت افزار + نصب + راه اندازی) | ≈ **۲۱٫۴** میلیارد تومان |
| توسعه پلتفرم (یک باره، قابل سرشکن در چند سایت) | ≈ **۷٫۷** میلیارد تومان |
| **مجموع کل** | ≈ **۲۹٫۱** میلیارد تومان |
| عملیات سال اول | ≈ ۱٫۶۵ میلیارد تومان |

هزینه توسعه یک باره است و برای ساختمان دوم عمدتاً فقط بخش استقرار مورد نیاز است.

| لایه | فناوری |
|---|---|
| سرویس های بک اند | **Go** |
| هوش مصنوعی / بینایی ماشین | **Python** + ONNX Runtime (TensorRT/CUDA) |

## مشاهده و توسعه محلی

نیازی به Build یا وابستگی نیست. فایل `index.html` (هاب) یا هر صفحه دیگر را مستقیم در مرورگر باز کنید.
صفحات تعاملی (ماشین حساب هزینه و ROI) تغییرات را در `localStorage` نگه می دارند و دکمه
**Reset** برای بازگردانی مقادیر پیش فرض دارند.

## 🛠️ پشته فناوری

### پشته پیشنهادی محصول
پلتفرم نگاهبان با فناوری های سازمانی زیر معماری شده است:

**بک اند و پردازش**
- **Go** - سرویس های بک اند با کارایی بالا و پردازش همزمان
- **Python** - خط لوله AI/ML با OpenCV و TensorFlow برای بینایی ماشین
- **ONNX Runtime** - استنتاج بهینه با شتاب TensorRT برای پردازش لحظه ای

**فرانت اند و کلاینت**
- **React + TypeScript** - داشبورد وب مدرن و Type-safe
- **Flutter** - اپلیکیشن های بومی iOS/Android
- **HTML5/CSS3** - رابط کاربری استاندارد وب

**داده و ذخیره سازی**
- **PostgreSQL** - پایگاه داده رابطه ای اصلی با pgvector برای embedding
- **TimescaleDB** - بهینه سازی سری زمانی برای لاگ رخدادها
- **MinIO** - ذخیره سازی آبجکت سازگار S3 به صورت On-Prem برای تصویر/ویدیو
- **Redis** - کش درون حافظه و رهگیری حضور لحظه ای

**یکپارچه سازی و پیام رسانی**
- **NATS JetStream** - گذرگاه پیام قابل اعتماد برای رویدادهای سیستم
- **GStreamer / FFmpeg** - پردازش و مدیریت استریم ویدیو

**زیرساخت و مشاهده پذیری**
- **Docker + Kubernetes** - ارکستریشن کانتینر و مقیاس پذیری
- **Prometheus + Grafana** - متریک و مصورسازی
- **Loki** - لاگ متمرکز توزیع شده
- **OpenTelemetry** - رهگیری انتها به انتها

### این مخزن
این مخزن یک **ارائه HTML ایستا** از پروپوزال نگاهبان است:
- بدون نیاز به Build یا Compile
- HTML/CSS خالص به همراه Chart.js برای مصورسازی تعاملی
- طراحی واکنش گرا با پشتیبانی فارسی (RTL) و انگلیسی (LTR)
- استقرار خودکار روی GitHub Pages

## 🚀 شروع سریع

### مشاهده آنلاین
برای مشاهده نسخه زنده، به این آدرس بروید:
[https://sobhanaz.github.io/Gardian-One/](https://sobhanaz.github.io/Gardian-One/)

### توسعه محلی
1. **کلون کردن مخزن:**
   ```bash
   git clone https://github.com/sobhanaz/Gardian-One.git
   cd Gardian-One
   ```

2. **باز کردن در مرورگر:**
   - با `index.html` به عنوان هاب ورودی شروع کنید
   - به هر سند از طریق لینک های داخلی بروید
   - نیازی به وابستگی، Build tool یا Local server نیست

3. **ویرایش و تست:**
   - هر فایل HTML را ویرایش کنید
   - مرورگر را Refresh کنید
   - صفحات تعاملی مقادیر را در `localStorage` نگه می دارند
   - با دکمه "Reset" مقادیر پیش فرض را بازگردانید

### ماشین حساب های هزینه و ROI
- **ماشین حساب هزینه:** ویرایش تعداد تجهیزات، نرخ های اجرا و زمان بندی
- **مدل ROI:** تنظیم فرضیات صرفه جویی و هزینه مبنا
- همه تغییرات به صورت محلی ذخیره می شوند؛ با بستن و بازکردن مرورگر قابل بررسی هستند
